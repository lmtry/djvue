# djvue ——适用于初学者的前后端分离教学
--------
## Django 和 Vue 实现前后端分离开发 使用Fetch API 进行前端AJAX 与后端服务器交互 
## 实现功能
* GET 和 POST 操作
## 技术内容
* Django
* Vue
* Fetch API
* rest framework
